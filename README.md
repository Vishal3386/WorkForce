# WorkForce Pro

**Expense & Workforce Management System**

A fully client-side web application for managing workforce, attendance, expenses, inventory, payroll, and more — built with vanilla HTML, CSS, and JavaScript.

## 📁 Project Structure

```
WorkForcePro/
├── index.html    # Main HTML structure
├── style.css     # All styles and CSS variables
├── app.js        # Application logic and data
└── README.md     # This file
```

## 🚀 Getting Started

Simply open `index.html` in any modern browser — no build tools or dependencies required.

```bash
# Clone the repo
git clone https://github.com/your-username/WorkForcePro.git

# Open in browser
open WorkForcePro/index.html
```

Or serve locally with any static server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## ✨ Features

- 👥 Employee management with detailed profiles
- 📋 Attendance tracking with QR check-in/out
- 💰 Expense tracking and payroll management
- 🏗️ Site & project management
- 📦 Inventory tracking and material transfers
- 📊 Revenue & partner management
- 📄 Document management
- ⚙️ Settings & data backup

## 🛠️ Tech Stack

- Pure HTML5, CSS3, JavaScript (ES6+)
- No frameworks, no dependencies
- Works offline, fully client-side
